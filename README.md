# Airplane-tickets-UA
***Магазин авиабилетов***

В качестве UI - **Asp.net Core**.

Рейсы генерируются за **90** дней. 
При генерации есть **10%** вероятность создания скидки **15%** на весь рейс. 
- За 60 дней до вылета генерируются специальное предложение - скидка **10%**. 
- За 30 дней - скидка **5%**. 
- За 1 день - скидка **50%**.
